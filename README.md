# XXMI Tools CHN
XXMI Tools修改版，添加一些小功能，方便Mod导出以及调试，仅在blender4.4测试。  
原神：  
锁定贴图插槽并自动应用ORFix  
补充原神hash风格模板文件  
补充原神hash和槽位风格混用模板文件，须自定义使用hash风格的部件名，参考如下：  
    {% set hash_style_parts = ["XianyunHead", "XianyunBody",  "XianyunDress"] %}    
如何使用自定义模板文件请自行学习  
ALL Games：  
自动填充Dump Folder路径  
保留形态键应用修改器（默认记录数值）  
快速重命名并自动排序顶点组  
deepseek翻译形态键并全局同步调整形态键  
骨骼生成器，快速根据权重生成基础骨骼  
ini文件自动生成[Constants]节  
导入模型时清理松散点及干扰顶点组  
导入模型时自动X轴镜像并翻转面朝向  
支持顶点组自动填充导出  
顶点色快速预设  
UV重命名  
锁定顶点权重
